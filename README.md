# PurpleMerit User Management System

This repository contains a full-stack user management system built as part of the PurpleMerit backend assessment.

## Structure

purplemerit-user-management/
├── backend/ # Node.js + Express + MongoDB API
├── frontend/ # React (Vite) frontend

## Backend Features

- JWT authentication
- Role-based access control (Admin / User)
- User profile management
- Password update
- Admin user management (activate / deactivate users)

## Frontend Features

- Login & Signup
- Protected routes
- User profile page
- Admin dashboard
